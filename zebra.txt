Commands start with a caret (^) 
Command names follow the caret are one or two letters long, and are case-insensitive
Commands can have a number of parameters
ommand parameters are separated by comma (,) characters
Whitespace (spaces, tabs, newlines) is mostly ignored; the main exception is the ^FD
^XA and ^XZ, serve as bookends to ZPL label definitions.
The ^FS command is used to finish the current field and start a new field.
